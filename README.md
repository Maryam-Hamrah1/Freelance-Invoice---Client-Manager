# Freelance Invoice & Client Manager

This is a simple web application for managing clients and invoices.  
It helps freelancers track their work, clients, and payments easily.

## Features
- Add and manage clients
- Create invoices
- View invoice list
- Simple and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Download or clone the project
2. Open index.html in your browser
